# Contributing

This repository is primarily intended to support reproducibility for the associated publication.

If you would like to propose improvements:

1. Open an Issue describing the problem (bug report, missing dependency, unclear documentation, etc.).
2. For code changes, open a Pull Request with:
   - A concise description of the change and why it is needed.
   - The exact command(s) used to test the change.

Please keep PRs focused and avoid large refactors unless required for correctness or reproducibility.
